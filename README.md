# Coffee-Shop-Sales

## Project Overview

This project focuses on analyzing coffee shop retail sales data to understand customer purchasing patterns, sales performance, and product popularity. The objective of this analysis is to extract meaningful insights from transactional data and identify opportunities to improve business performance.
Using data analysis techniques, the project evaluates how sales change across different days, hours, locations, and product categories.
